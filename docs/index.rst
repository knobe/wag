wag
===

===========
What is wag
===========

wag is program much like tail, but it works for your rss feeds.  And since its
supposed to be a fun tail it has some added features.

--------
Features
--------

 * Display any feed how you want using jinja2 templating
 * Display your feed in a terminal
 * Constantly watch a certain feed
 * Get certain feeds after a certain *date*


=======
Install
=======

To install wag use pip, it will make you happy.

    [sudo] pip install wag

If you want easy_install works too.

    [sudo] easy_install wag
